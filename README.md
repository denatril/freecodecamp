# freeCodeCamp Web Development Labs

This repository contains a collection of mini-projects and workshops completed as part of the freeCodeCamp curriculum. These labs focus on HTML5 structure, multimedia integration, and responsive web design.

## 🚀 Projects Included

### 1. HTML Audio and Video Player
A media-focused page demonstrating how to embed and control native HTML5 video and audio elements.
*   **Key Features:** Custom video dimensions, media controls, and looped audio playback.
*   **Technologies:** HTML5 `<video>`, `<audio>`, and `<source>` tags.

### 2. Travel Agency Landing Page
A promotional landing page for a travel agency specializing in tours of Turkey and Italy.
*   **Key Features:** Semantic structure, SEO description tags, image figures with captions, and external resource linking.
*   **Technologies:** HTML5 `<figure>`, `<figcaption>`, and descriptive metadata.

### 3. Video Compilation Page
A tutorial-style page ("3 Ways to Cook an Omelette") that aggregates external video content.
*   **Key Features:** Embedding third-party content (YouTube) using iframes and organized content sections.
*   **Technologies:** HTML5 `<iframe>`, `<section>`, and `<main>`.

### 4. Event Hub
A structured community board displaying upcoming and past professional events.
*   **Key Features:** Internal navigation (anchor links), article-based content layout, and event archiving with descriptive imagery.
*   **Technologies:** HTML5 `<nav>`, `<article>`, and accessibility-focused `alt` text.

## 🛠 Setup & Usage

To view any of these projects:
1. Clone the repository:
   ```bash
   git clone https://github.com/denatril/freecodecamp.git
   ```
2. Navigate to the specific project folder or file.
3. Open the `index.html` file in any modern web browser.
